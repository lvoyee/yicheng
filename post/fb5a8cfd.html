基于Spring Boot后台管理系统

codeshare spring boot 2020/02/15

叨叨叨

spring boot 后台管理系统，可以让你接触到的技术栈有以下：

  • spring boot
  • shiro
  • druid
  • maven
  • …

数据库用的是 mysql，前端使用的是 thymeleaf模板引擎。
开发工具：IntelliJ IDEA ，JDK 1.8

现在已经实现的功能点有：

  • 登录 / 退出
  • 用户管理
  • 角色管理
  • 权限管理

基本上学习用户角色权限，作为入门就足够了哦！如果有其他的想法，你可以下载这个代码根据自己的需求加上你自己的需求即可。
