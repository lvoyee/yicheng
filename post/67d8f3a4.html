基于SpringBoot开发的医疗管理系统源码分享

codeshare 医疗管理系统 医院管理后台系统 2023/09/15

写在前面

本项目是基于Spring Boot 2.x 开发的医疗管理系统系统。本项目也可以当作毕业设计，期末课程作业等，也可以当作学习、进阶Spring Boot 的资料。

功能描述

本项目主要包含以下功能模块

  • 系统管理
  • 医生管理
  • 患者管理
  • 药品管理
  • 管理员管理
  • 预约管理
  • 病史管理
  • 住院信息管理
  • 登录/注销功能
  • …

开发环境（运行环境）

  • 系统环境：Windows 11
  • 开发工具：IntelliJ IDEA 2023.1.2
  • Java版本：JDK 1.8
  • Mysql版本：8.0
  • Maven版本：3.6.3

项目技术栈

  • Spring Boot 2.1.4.RELEASE
  • Mybatis
  • Maven 3.X
  • Mysql
  • layui
  • Jquery
  • freemarker
  • …
