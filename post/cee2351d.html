基于Spring Boot开发的新闻管理系统源码分享

codeshare 新闻管理 源码 2024/05/01

写在前面

本项目是基于Spring Boot 2.x 开发的新闻管理系统系统，系统主要分为前台浏览页面和后台管理页面。本项目也可以当作毕业设计，期末课程作业等，也可以当作学习、进阶Spring Boot 的资料。

功能描述

本项目主要包含以下功能模块

  • (前台)登录/注册
  • (前台)用户中心
  • (前台)发布新闻
  • (前台)评论、收藏新闻
  • (前台)评论敏感词过滤
  • …
  • (后台)新闻分类管理
  • (后台)新闻审核管理
  • (后台)新闻发布管理
  • (后台)新闻评论管理
  • (后台)用户管理
  • (后台)登录/登出
  • …

开发环境（运行环境）

  • 系统环境：Windows 11
  • 开发工具：IntelliJ IDEA 2023.1.2
  • Java版本：JDK 1.8
  • Mysql版本：8.0
  • Maven版本：3.6.3

项目技术栈

  • Spring Boot 2.X
  • Mybatis-Plus
  • Maven 3.X
  • Mysql
  • layui
  • Jquery
  • freemarker
  • …
